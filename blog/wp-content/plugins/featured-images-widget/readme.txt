Display featured images from recent posts in a sidebar widget.

== Description ==

Show your featured images from recent posts in a sidebar widget. Filter by category and select the amount of images you would like to show.

== Installation ==

1. Upload folder to wp-content/themes
2. Go to Appearance > Widgets
3. Drag "Featured Images Widget" into the sidebar where you want to display it
 
== Frequently Asked Questions ==

= Can I change the sort order of the images? =

Not yet.

= Why aren't my thumbnails loading in the widget? =

Check your posts and make sure you have the image set as a featured image.
